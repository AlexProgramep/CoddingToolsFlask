git init

Создание локального репозитория и папки .git

git config

Используется для указания пользовательских настроек, таких как электронная почта, имя пользователя, формат и т.д.

git add

Используется для добавления файлов в индекс.

git clone ссылка

клонируется репозиторий в папку.

git commit -m "Первый коммит"

Создание коммитов(комментариев) в проекте.

git push origin название_ветки

Отправка проекта в github.

git pull

Вытягивание изменений с github репозитория.

git checkout b название_ветки

Создание веток и/или переключение между ними.

git remote -v

Для отображения списка подключенных репозиториев github

git remote add origin путь_к_папке

Позволит пользователю подключить проект к репозиторию github.

git branch

Отображение всех существующих веток в репозитории.

git branch –d название_ветки

Удаление ветки.

git branch -M название_ветки

Создание ветки.

git merge название_ветки

Объединение ветки в активную ветвь.

git branch -M название_ветки

Создание ветки.

git diff

Отображения существующих различий.

git diff

Отображения существующих различий.

git diff --base путь_к_папке

Выявления различий с базовыми файлами.

git diff ветвь_источник ветвь_цель

Используется для просмотра различий между ветками, которые должны быть объединены, до их объединения.

git tag версия(например 1.1.0) commitID

Маркировка определенных коммитов с помощью простых меток.

git diff

Отображения существующих различий.

git log

Отобразит список всех коммитов в ветке вместе с соответствующими сведениями.

git reset --hard каталог

Сброс индекса и рабочего каталога до последнего состояния коммита.

git rm имяфайла.расширение

Удаления файлов из индекса и рабочего каталога.

git stash

Помогает в сохранении изменений на временной основе, эти изменения не попадут в коммит сразу.

git show

Для просмотра информации о любом git объекте.

git fetch origin

Позволяет пользователю доставить все объекты из удаленного репозитория, которые не присутствуют в локальном рабочем каталоге.

gitk

Графический интерфейс локального репозитория.
